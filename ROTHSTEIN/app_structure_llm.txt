## Environment Variables Required
- PORT (default: 8080)
- MODE (PAPER or LIVE)
- DASHBOARD_PASSWORD (default: 7777777lL)
- DATA_DIR (/app/data)
- POLY_API_KEY (LIVE mode only)
- POLY_API_SECRET (LIVE mode only)
- POLY_PASSPHRASE (LIVE mode only)
- POLY_WALLET_ADDRESS (LIVE mode only)
- POLY_PRIVATE_KEY (LIVE mode only)

## Key Config Values
- Scan interval: 15s
- Min trade score: 60
- Price range: 0.45-0.85
- Edge minimum: 0.05
- Secs remaining: 90-300
- Dead hours: [15, 16] UTC (configurable from dashboard)
- Sizing tiers: $5 (score 60) to $20 (score 90+)
- 12 tracked whale wallets (direct polling via Polymarket Data API)
- Whale poll interval: 5s (staggered across wallets)

## Notes
- Region: Amsterdam (europe-west4) — Binance WS works directly, Polymarket via Irish IP
- Polymarket WS requires text "PING" every 10s (not protocol-level ping)
- Contract scanner uses /markets endpoint with slug pattern (btc|eth)-updown-(5m|15m)-{ts}
- Dashboard password-protected with token-based auth
- Conditional TP based on market price (polyMid >= 0.85) + edge drop below 0
- Dashboard is the primary control interface — dead hours, pause, scoring all configurable
- PAPER mode simulates trades; LIVE mode requires CLOB credentials
