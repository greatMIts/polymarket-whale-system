Layer 0: types.ts, config.ts, logger.ts, persistence.ts
Layer 1: binance-feed.ts, polymarket-book.ts, contract-scanner.ts, whale-listener.ts
Layer 2: pricing.ts, features.ts, scorer.ts, pnl.ts
Layer 3: decisions-log.ts, risk.ts, positions.ts, trader.ts
Layer 4: scanner.ts
Layer 5: server.ts, index.ts
Dashboard: dashboard/index.html

## Environment Variables Required
- PORT (default: 3334)
- MODE (PAPER or LIVE)
- SPY_SERVER_URL (ws://polymarket-whale-bot.railway.internal)
- DATA_DIR (/app/data)
- POLY_API_KEY (LIVE mode only)
- POLY_API_SECRET (LIVE mode only)
- POLY_PASSPHRASE (LIVE mode only)
- POLY_WALLET_ADDRESS (LIVE mode only)
- POLY_PRIVATE_KEY (LIVE mode only)

## Key Config Values
- Scan interval: 15s
- Min trade score: 60
- Price range: 0.45-0.85
- Edge minimum: 0.05
- Secs remaining: 90-300
- Dead hours: [15, 16] UTC (configurable from dashboard)
- Sizing tiers: $5 (score 60) to $20 (score 90+)

## Notes
- Binance WS blocked in US (451). Auto-failover to data-stream.binance.vision
- Conditional TP based on market price (polyMid >= 0.85) + edge drop below 0
- Dashboard is the primary control interface — dead hours, pause, scoring all configurable
- PAPER mode simulates trades; LIVE mode requires CLOB credentials
